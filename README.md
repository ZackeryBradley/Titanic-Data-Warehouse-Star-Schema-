# Titanic Data Warehouse (Star Schema)

## 📌 Overview
This project designs a star schema data model using Titanic dataset features to support analytical queries and reporting.

---

## ❓ Business Problem
How can we structure raw data into a scalable model for fast and efficient analytics?

---

## 🛠 Tools Used
- Python (Pandas)
- Data Modeling
- Star Schema Design

---

## 🧱 Data Model
- Fact Table: Passenger survival and attributes  
- Dimension Tables:
  - Passenger demographics
  - Ticket information
  - Class / cabin data  

---

## 📊 Key Outcomes
- Built a clean star schema structure for analytics  
- Optimized dataset for reporting queries  
- Improved query clarity and scalability  

---

## 🚀 Why This Matters
Star schema design is critical in analytics engineering and BI environments for performance and usability.
